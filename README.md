# 📝 Todo App

## ✨ Features
- Add, complete, and delete tasks
- Filter tasks (All/Active/Completed)
- Dark/Light mode
- Mobile-friendly
- Data persists in local storage

## 🚀 Quick Start
```bash
git clone https://github.com/XORO1337/todo-app.git
cd todo-app
npm install
npm start
```
